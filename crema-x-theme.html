Worked Files

  • custom.css 📝

    This file contains custom styles that were added to enhance the overall aesthetics of the website. Changes include adjustments to typography, colors, and layout elements for better alignment with the brand's identity.

  • section-image-banner.css 📝

    Styles for the image banner section were updated to improve responsiveness and visual appeal. New hover effects and transitions were added for a smoother user experience.

  • section-image-banner.js 📝

    JavaScript functionality was implemented to enable dynamic content switching in the image banner. This enhances user engagement by displaying multiple images seamlessly.

  • component-card.css 📝

    The card component styles were revised to ensure consistency across the site. Modifications include padding adjustments and border-radius changes for a more modern look.

  • base.css 📝

    Base styles were adjusted to create a cohesive design foundation. This includes resetting default browser styles and setting a consistent color palette throughout the site.

  • component-image-with-text.css 📝

    Updates to this component focused on improving text visibility over images. Added text shadows and background overlays to enhance readability and visual hierarchy.

  • component-image-with-text-2.css 📝

    This file includes additional styles for a variation of the image-with-text component. New layout options were added to support different content arrangements.

  • custom-product-variant.css 📝

    Custom styles were created for product variants to improve the presentation and functionality of product selection. This includes better alignment of variant options and enhanced hover effects.

  • component-product-variant-picker.css 📝

    Styles for the product variant picker were refined to create a more user-friendly interface. Changes include improved spacing and visibility for selected options.

  • theme.liquid 📝

    The main theme file was modified to integrate new components and styles. This includes adding conditional logic for displaying different layouts based on user interactions.

  • en.default.schema.json

    Updates to the schema file were made to include additional settings for new features. This ensures that the theme remains compatible with future updates.

  • header.liquid 📝

    The header was redesigned to include a more streamlined navigation experience. Changes also included adding a search bar and social media links for better accessibility.

  • image-banner.liquid 📝

    The image banner section was revamped to allow for more customization options. Modifications include enabling the use of different image sizes and formats.

  • featured-collection.liquid 📝

    Changes to the featured collection section enhanced the layout and visibility of products. Updated styles ensure that products are displayed attractively and responsively.

  • main-product.liquid

    The main product template was updated to include new product information sections. Enhanced layout for better user experience and information clarity.

  • image-with-text.liquid

    Enhancements were made to this section for better integration of images with text. New features include adjustable image sizes and alignment options.

  • custom-basic-page.liquid

    A new template was created to support basic content pages with flexible layout options. This allows for more straightforward content management.

  • contact-form.liquid

    The contact form was updated to improve user submission experience. Added validations and styling enhancements ensure that the form is both functional and visually appealing.

  • card-collection.liquid 📝

    The styles and layout of the card collection were modified to present collections more attractively. New hover effects and improved spacing enhance the visual experience.

  • icon-account.liquid

    The account icon was redesigned for a fresher look. This change aligns with the overall design updates throughout the site.

  • popup-product-details.liquid

    The popup for product details was improved to display more information clearly. Enhanced styles and animations were added for a smoother user experience.

  • header-search.liquid

    The header search functionality was improved with better styles and integration into the main header. Ensures users can easily find products.

  • icon-hamburger.liquid

    The hamburger menu icon was updated for better visibility and interaction. Changes include size adjustments and hover effects.

  • product-variant-options.liquid

    This snippet was revised to streamline the presentation of product variants. Adjustments were made to enhance usability and visual clarity.

  • page.custom-basic-template.json

    The custom basic template was created to facilitate the use of standard page layouts. It supports various content types and enhances site flexibility.

Product Metafields

Terms and Conditions

Here you can add your terms and conditions relevant to this product. Ensure clarity and completeness to inform customers about their purchase.

What You Can

Details about what the customer can expect from the product, its usage, and features can be added here. Use the rich text editor for formatting.

Includes

List what is included with the product. This might be accessories, components, or services provided with the purchase.

What You Need

Specify any requirements or additional items needed for the product to function properly, such as batteries, software, or other equipment.
